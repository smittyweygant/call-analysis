## Evaluation Criteria

### Technical Competency (Weight: 40%)
| Rating | Description |
|--------|-------------|
| Strong | Demonstrates deep expertise, can teach others |
| Good | Solid understanding, can work independently |
| Developing | Basic knowledge, needs mentorship |
| Gap | Significant learning required |

### Problem Solving (Weight: 25%)
| Rating | Description |
|--------|-------------|
| Strong | Structured approach, considers edge cases |
| Good | Logical thinking, reaches solutions |
| Developing | Needs guidance on approach |
| Gap | Struggles with problem decomposition |

### Communication (Weight: 20%)
| Rating | Description |
|--------|-------------|
| Strong | Clear, concise, adapts to audience |
| Good | Effective communication |
| Developing | Sometimes unclear or verbose |
| Gap | Difficulty conveying ideas |

### Culture Fit (Weight: 15%)
| Rating | Description |
|--------|-------------|
| Strong | Embodies company values |
| Good | Aligns with team culture |
| Developing | Some alignment concerns |
| Gap | Significant culture mismatch |
